# hnefatafl-platform

Scalable online Hnefatafl platform: real-time human-vs-human play with clocks, matchmaking, and basic game history.

## MVP scope
- Clerk auth
- Lobby + matchmaking by time control
- Real-time games (WebSocket)
- Server-authoritative rules + clocks
- Rematch + basic game history
